### Strings and String Literals
- A **string** is a sequence of characters
- A **string literal** surrounds a character sequence with double quotes as in "Hello" vs. an integer literal like 42 or character literal like 'a'

---
### Getting a String without Whitespaces from Input
- A **whitespace character** is a character used to represent horizontal and vertical spaces in text, and includes spaces, tabs, and newline characters

---
### Getting a String with Whitespace from Input
- The function **getline** (cin, stringVar) gets all remaining text on the current input line, up to the next newline character
