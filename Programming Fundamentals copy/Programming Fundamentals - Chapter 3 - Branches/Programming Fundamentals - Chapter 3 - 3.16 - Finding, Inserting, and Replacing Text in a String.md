### Finding Text in a String
- The **find()** function is used to find a string or character inside a string
- **string::npos** is a special value representing "no position"
---
### Getting a Substring
- A **substring** is a contiguous part of a string
- The **substr()** function is used to get a substring of a string
---
### Inserting and Replacing Text in a String
- The **insert()** function inserts text into a string
- The **replace()** function replaces text in a string
