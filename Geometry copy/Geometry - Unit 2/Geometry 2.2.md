1. The set of *consecutive integers* is {...-3, -2, -1, 0, 1, 2, 3...}
2. The set of *even integers* is {...-6, -4, -2, 0, 2, 4, 6...}
3. The set of *odd integers* is {...-5, -3, -1, 1, 3, 5...}
4. When many observations lead to the same discovery, a person could use **Inductive Reasoning** to make a **conjecture**, which is an unproven statement based upon those observations

**Postulates that we need to know**
- Through any two points, there is exactly one line
- Through any three points, there is at least one plane (collinear - inf many planes) (NOT collinear - no inf many planes)
- If two lines intersect, then their intersection is a point
- If two planes intersect, then their intersection is a like (think of where the wall and ceiling meet)
- Through a line and a point not in the line there is exactly one plane
- If two lines intersect, then exactly one plane contains the line
