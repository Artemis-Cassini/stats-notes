1. A tool is used to construct circles is a compass
2. A tool used to construct straight lines is called a straight edge
3. A tool used to measure the length of a line segment is a ruler
4. A tool used to measure the size of an angle in degrees is a protractor

**The 3 Undefined Terms in Geometry**
5. A point is the simplest figure studied in geometry
- It is undefined, although we can describe it
- It has no size
- It is infinitely small
- It has only location
- A very sharp pencil tip is a physical model of a point
- A point, however, is smaller than the smallest dot you can ever make with a pencil

6. A line is a straight arrangement of points
- It is undefined, although we can describe it
- There are infinitely many points in a line
- It has length but no width
- It extends forever in two directions without ending
- A taut string is a model of a line
- A line, however, is thinner than any string every made

7. A plane has length and width but no thickness
- It is undefined, although we can describe it
- It is a flat surface that extends forever
- A piece of flat cardboard is a model of a plane
- A plane, however, is thinner than any cardboard ever made

8. The three undefined terms in Geometry are the point, line, and plane

**Defined Terms**
9. The set of all points is called space
10. When points lie on the same line, they are called collinear 
11. When points lie in the same plane, they are called coplanar 

**Naming Basic Geometric Objects**
12. A point is named with a single capital letter
13. A line segment is named with two capital letters with a segment symbol above the two letters
14. A line is named with two capital letters with a line symbol above the two letters
15. A line may also be named using a single lowercase letter
16. A ray is named with two capital letters with a ray symbol (pointing to the right) above the two letters. The first letter is the endpoint of the ray, while the second letter is any other point on the ray. 
17. An angle is named with:
- an angle symbol followed by three capital letters; the vertex of the angle must be on the middle letter
- a single capital letter as long as there are not multiple angles located at that same vertex
- a number by the vertex and within the interior of the angle
18. A polygon can be named by picking any vertex, writing down the name of that point, followed by the names of all the other vertices in order - either clockwise or counterclockwise 
19. A plane can be named the same way as a polygon OR a plane can be named using a single capital letter shown within the interior of the parallelogram representing the plane
