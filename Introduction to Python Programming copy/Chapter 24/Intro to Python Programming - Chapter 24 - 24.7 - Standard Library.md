- The **Python standard library** includes various utilities and tools for performing common program behaviors

| Module Name | Description                                                                                |
| ----------- | ------------------------------------------------------------------------------------------ |
| datetime    | Functions for creating and editing of dates and times objects                              |
| random      | Functions for working with random numbers                                                  |
| copy        | Functions that create complete copies of objects                                           |
| time        | Functions that get the current time, convert time zones, and sleep for a number of seconds |
| math        | Mathematical functions                                                                     |
| os          | Operating system informational and management helper functions                             |
| sys         | System-specific environment of configuration helper functions                              |
| pdb         | The Python interactive debugger                                                            |
| urllib      | URL handling functions, such as requesting web pages                                       |
