**Variables and Assignments**
- A variable is a named item, like x or num_people, it holds a value
- An assignment statement assigns a variable with a value, like x=5, this means that x is assigned with 5, so x will keep that value during subsequent statements

**Assignments with Variable on Left and Right**
- Since = means assignment in programming, a variable may appear on both sides, such as x = x + 1, if 6 was 6, x is assigned with 6 + 1, or 7
- You can increment the value as well
