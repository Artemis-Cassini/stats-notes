**Floating-Point Numbers and Scientific Notation**
- A floating-point number is a real number such as 98.6 or 0.001, the term refers to the decimal point appearing anywhere in the number
- A float is a data type for floating-point numbers
- A floating-point literal is written with the fractional part even if that fraction is 0, as in 1.0, 0.0 or 99.0
- Scientific Notation is when you use an 'e' preceding the power of 10 exponent 

**Overflow**
- Assigning a floating-point value outside of the range generates an OverflowError

**Manipulating Floating-Point Output**
- You can manipulate one with .2f
