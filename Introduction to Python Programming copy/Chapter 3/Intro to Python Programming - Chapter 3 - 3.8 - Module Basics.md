**Modules**
- Programmers typically write Python program code in a file called a script, and execute the code by passing the script as input to the Python interpreter 
- A module is a file containing Python code that can be used by other modules or scripts
- A module is made available for use via the import statement
- Once is it imported, any object defined in that module can be accessed using dot notation 

**Importing Modules and Executing Scripts**
- Python programs often use the built-in special name __ name __ to determine if the file was executed as a script by the programmer or imported by another module
