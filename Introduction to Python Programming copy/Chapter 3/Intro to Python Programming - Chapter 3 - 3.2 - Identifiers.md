**Identifier Rules**
- An identifier, or a name, is a sequence of letters, underscores, and digits, and must start with a letter or underscore
- Python is case sensitive, wich means uppercase v lowercase letters can differ
- Certain words cannot be used as names, reserved words, or keywords, are words that are part of the language and cannot be used as a programmer-defined name
