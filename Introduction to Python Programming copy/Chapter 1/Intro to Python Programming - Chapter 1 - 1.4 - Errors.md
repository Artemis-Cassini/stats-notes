**Syntax Errors**
- As soon as a person begins trying to program, that person will make mistakes. One kind of mistake, a syntax error, violates a programming language’s rules on how symbols can be combined to create a program 

**Runtime Errors**
- A runtime error is where a program’s syntax is correct but the program attempts an impossible operation 

**Logic Errors**
- A logic error is when the code is mistyped, or a bug
