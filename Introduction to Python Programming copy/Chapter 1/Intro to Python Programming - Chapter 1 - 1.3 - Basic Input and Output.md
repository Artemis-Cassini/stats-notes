**Basic Text Output**
- Text is output from a Python program using the built-in function print()
- Text enclosed in quotes is known as a string

**Output Variables and Multiple Values**
- The value of a variable can be printed out via: print(variable_name) 
- A single output line may include multiple strings and variables separated by commas

**Keeping Output on the Same Line**
- Each use of print() outputs on a new line
- You can also use the newline character
- An escape sequence is a string that has a special meaning
- Any space, tab, or newline is called whitespace

**Basic Input**
- The input() function is used to read input from a user
