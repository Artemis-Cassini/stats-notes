**Strings and String Literals**
- A string is a sequence of characters that represents textual data like a person’s name, location, or a message to the user
- One can be created from a user input, or being defined in the source code
- A string literal is a string defined in the source code of a program by surrounding the text value with single or double quotes, such as ‘Poppy’ or “Poppy”
- This is known as a sequence type, a type that orders a collection of objects in a sequence from first to last
- A common operation to find the length, or the number of characters in a string. The *len()* built-in function can help to find the length of the string

**String Indexing**
- A programer can reference a character at a specific index by appending brackets **[]** containing the desired index after the name of a string value

**Changing String Variables**
- Strings cannot change once created, they are immutable ojects
- TypeErrors will appear when executed 

**Concatenating Strings**
- Programs can add new characters to the end of the string, which is known as string concatenation
- This can only include string objects, otherwise attempting this with a number and string will result in a TypeError
