**Creating a List**
- A container is a construct used to group related values together and contains references to other objects instead of data
- A list is a container created by surrounding a sequence of variables or literals with brackets
- A list item is called an element, it is mutable, meaning that the elements in a list can be replaced, reordered, and or removed
- A list is also a sequence type, meaning the contained elements are ordered by position in the list, know as the element's index

**Accessing List Elements**
- Lists are useful for reducing the number of variables in a program
- Instead of having a separate variable for the name of every one in a class or email, a single list can store an entire collection of related variables
- Individual list elements can be accessed using an indexing expression by using brackets as in my_list[i], where i is an integer

**Updating List Elements**
- Lists are mutable, which means they can change elements

**Adding and Removing List Elements**
- A method instructs an object to perform some action and is specified by providing the object name followed by a "." symbol and then the method name
- You can use append(), pop(), or remove()

**Sequence-Type Methods and Functions**
- Sequence-type methods are functions built into the Python language definitions of sequence types like lists and strings
- Some examples are append(), pop(), and remove()
