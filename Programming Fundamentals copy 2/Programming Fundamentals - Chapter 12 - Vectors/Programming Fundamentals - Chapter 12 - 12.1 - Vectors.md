### Vector Declaration and Element Access
- A **vector** is an ordered list of items of a given data type
- Each item in a vector is called an **element**
- **push_back()** appends a new element to the end of an existing vector
- **at()** accesses a vector element at the index specified within the parentheses
--- 
#### Curly Braces and Angle Brackets
- {} are **braces**
- < > are **angle brackets**, or **chevrons**
----
### Loops and Vectors
- A vector's **size()** function returns the number of vector elements
