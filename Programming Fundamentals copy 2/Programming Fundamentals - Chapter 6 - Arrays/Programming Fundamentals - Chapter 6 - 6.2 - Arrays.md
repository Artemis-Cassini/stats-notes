### Arrays Declarations and Accessing Elements
- An **array** is an ordered list of items of a given data type
- Each item in an array is called an **element**
- [] are **brackets**
- {} are **braces**
- The number in the brackets is called the **index** of the corresponding element
---
### Array Initialization
- Like other variables, the keyword **const** may be prepared to an array variable declaration to prevent changes to the array
