### Pointers and Dynamically Allocated Arrays
- A **dynamically allocated array** is an array whose size and memory location are determined during runtime
- A **pointer** is a variable that holds a memory address, rather than holding data like most variables
- The **new operator** allocates memory for the given type and returns a pointer to the allocated memory
---
### Deleting Dynamically Allocated Array
- The **delete[] operator** is used to deallocate an array allocated with the new operator
----
### nullptr
- **Null** means "nothing" 
- A pointer that is assigned with the keyword **nullptr** is said to be null
