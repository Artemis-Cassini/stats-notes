### Pointer and Dynamically Allocated Objects
- A **pointer** is a variable that holds a memory address, rather than holding data like most variable
- The **new operator** allocates memory for the given type and returns a pointer to the allocated memory
---
### Member Access Operator
- When using a pointer to an object, the **member access operator (->)** allows access to the object's members with the syntax
---
### Dereference Operator
- The **dereference operator (•)** is prepared to a pointer variable's name to retrieve the data to which the pointer variable points
---
### Deleting Dynamically Allocated Objects
- The **delete operator** deallocates a block of memory that was allocated with the new operator
