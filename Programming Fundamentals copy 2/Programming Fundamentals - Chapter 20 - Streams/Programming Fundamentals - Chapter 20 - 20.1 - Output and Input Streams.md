### The ostream and cout streams
- An **ostream**, short for "output stream", is a class that supports output
- ostream provides the **<< operator**, known as the **insertion operator**
- **cout** is a predefined ostream object
---
### The istream and cin streams
- An **istream**, short for "input stream", is a class that supports input
- istream provides the **>> operator**, known as the **extraction operator**, to extract data from a data buffer and write the data into different types of variables.
- **cin** if the predefined istream pre-associated with a system's standard input
