### String Character Indices
-  Each string character has a position number called an **index**, starting with 0 (not 1)
---
### Accessing String Characters
- **at()** - The notation someString.at(x) accesses the character at index x of a string
---
### Working with the End of a String
- The function s1.**size()** returns s1's length
- **push_back()** adds a character to the end of a string
- **append()** adds one string to the end of another
- The **+** operator can return a new string with one string appended to another string
---
### Common Errors
- An **exception** is a detected runtime error that commonly prints an error message and terminates the program
