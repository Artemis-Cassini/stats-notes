- Including the **cctype library** provides access to several functions for working with characters

| Character Function | Description                    |
| ------------------ | ------------------------------ |
| **isalpha**        | true if alphabetic: a-z or A-Z |
| **isdigit**        | true if digit: 0-9             |
| **isspcae**        | true if whitespace             |
| **toupper**        | Uppercase version              |
| **tolower**        | Lowercase version              |
