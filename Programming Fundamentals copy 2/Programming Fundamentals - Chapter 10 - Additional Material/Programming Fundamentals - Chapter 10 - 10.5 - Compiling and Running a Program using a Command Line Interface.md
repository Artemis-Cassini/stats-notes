### Compiling a Program using a Command-Line Interface
- A **command-line interface (CLI)** is a program that enables text commands to be input , often via the keyboard, and submitted to an operating system.
- CLIs are older than more commonly used **graphical user interfaces (GUI)**, which allow users to interact with operating systems using graphical icons, often via the mouse or touchpad
- A **source file** contains code that can be compiled into an executable program
- **g++** is a CLI command used to compile a C++ source file
--- 
### Running Executable Program File a.out
- **a.out** is the default name of an executable program file created by compiling a source file. The default location for a.out is in the same directory as the corresponding source file.
---
### Naming an Executable Program File
- Using g++ with the flag **-o** and a filename assigns that filename to the resulting executable file
