### Major Generative AI Models
- A **transformer** is a type of neural network that processes information using an attention mechanism. This mechanism weighs the importance of each piece of information in the context of the input and is run on a massively parallel scale to transform input into output. Transformers are developed initially for natural language processing, but later expanded to applications such as image understanding. Ex: ChatGPT and LLaMA are transformer-based models.
- A **diffusion model** learns image creation by first reducing a real image to noise step-by-step, then reversing the process to re-create the image. Once trained on a huge volume of image data, the model is able to create images from noise. Diffusion models can generate varied outputs like text, images, or even videos. Image generation is the most common application of the model as of the writing of this text.
- A **multimodal AI model** handles information of multiple types, or modalities. Ex: An AI model takes an image and an instruction as input and modifies the image according to the instructions. Multimodal AI models are built on core generative AI models, such as transformers for text and diffusion models for producing realistic images, videos, and audio.

### Prompting
- A **prompt** is the text input provided to a generative AI model to guide the model to produce desired output. A prompt may include instructions, context, and examples. The study of how to write effective prompts is called **prompt engineering**
---
### AI Chatbot
- An **AI chatbot** is software that emulates human conversation using AI
- A **request** can be a question, a task defined by goals, or a set of instructions sent to a system, such as the AI model that powers an AI chatbot.
- Upon receiving a request, the system returns a **response** that addresses the request.
- A **conversation** consists of a sequence of alternate requests and responses between two parties.
- Within a conversation, the requests and responses prior to the current request form the **context** of the current request.
